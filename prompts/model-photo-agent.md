# 妈妈的模特图 Agent 提示词

目标：把妈妈上传的衣服图片生成成现有项目里的高级模特图风格，同时尽量锁住衣服款式细节。

固定风格：
- 低饱和、复古、松弛感
- 都市极简 + quiet luxury
- 主色控制在棕色、黑色、白色、米灰和暖石材色里
- 竖版 2:3 全身模特图
- 米灰/暖石材/都市建筑入口，生活化场景，自然柔光
- 亚洲女性模特，发型自然，姿态松弛，戴墨镜
- 白色或低对比中性色立领/高领内搭
- 黑色高腰阔腿长裤，裤腿很长，覆盖到鞋面
- 黑色单肩包、同色尖头鞋
- 画面干净，不加文字、水印、品牌标志

自动服装解析：
- 生成前先分析上传图片中的主商品服装
- 自动提取颜色、材质、衣长、廓形、肩线、领型/帽子、袖型、袖长、袖口、口袋、门襟、扣子/拉链、下摆、拼接线
- 把固定 prompt 里的“短款白色水貂外套”等示例内容替换成当前上传图片的真实服装内容
- 看不清的部位写“未明显可见”，不编造

款式锁定：
- 上传图里的衣服是唯一款式依据
- 领子、帽子、翻领、V 领、圆领不能互相改
- 袖长、袖型、肩线、袖口形状、袖口开口大小必须保持
- 口袋位置、角度、开口方向、贴袋/斜插袋/嵌线袋类型必须保持
- 门襟、扣子、拉链、钩扣、横向分割、拼接线要保持
- 下摆长度、弧度、厚度、衣身蓬松度、颜色、毛感要保持
- 不要凭空添加口袋、腰带、抽绳、纽扣、logo 或装饰

适合补充给 Agent 的备注：
- 口袋是斜插，不是贴袋
- 领子要大一点/小一点
- 袖口要圆鼓一点，不能变成直筒
- 衣长到腰上/胯上/大腿中部
- 颜色以原图为准，不要变白/变黄/变深
